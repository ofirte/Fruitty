# Fruitty

My Fruittiy App.

## Installation

1. Install MongoDB from here
```bash
https://www.mongodb.com/try/download/community?tck=docs_server
```
2. install client dependencies with:
```
'npm install'
```
3.Install server dependencies with:
```
npm install express
npm install body-parser
```
## Usage
1. Run
```
node server.js
```
2. Go to ./clent and run
```
npm start
```
