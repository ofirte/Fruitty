# Fruitty

My Fruittiy App.

## Installation

1. Install MongoDB from here
```bash
https://www.mongodb.com/try/download/community?tck=docs_server
```
2. Initate server with:
```
node server
```
3. Initate client with:
```
'npm install'
```
## Usage
1. Run
```
nose server.js
```
2. Go to ./clent and run
```
npm start
```
